## üá∫üá∏ Project Description

**RowMaster** is a free web-based tool for crafters (knitting, crochet, embroidery) that helps you easily track your progress on any pattern or photo.

### Key Features:
* **Movable Guide Line:** A high-contrast horizontal line to keep you on the right row.
* **Drawing Tools:** Mark finished sections or highlight details directly on your pattern.
* **Text Notes:** Leave reminders or row counts anywhere on the screen.
* **Zoom & Rotate:** Easily work with fine details on mobile devices or tablets.
* **Save & Resume:** Export your project as a file and pick up exactly where you left off later.
